# Victor Vector
A vector sequencer for Norns inspired by [D:Machinery](https://dmachinery.wordpress.com/2013/01/05/the-vector-sequencer/)
## Controls
### Encoders
- ENC1 = Change Pages
- ENC2 = Scroll params
- ENC3 = Change param value
### Keys
- K2 = Toggle cell state
- K3 = Reset start point
### Grid
- Play/Pause: (1,1)
- Reset Playback: (1,3)
- Shift: (1,5)

The 5x5 grid is used to select a cell in order to change its parameters on the Norns display.
