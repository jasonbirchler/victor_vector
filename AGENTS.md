# Project Name

This project is a vector sequencer written in Lua for the [Monome Norns](https://monome.org/docs/norns/). It is inspired by a post on the blog [D:Machinery](https://dmachinery.wordpress.com/2013/01/05/the-vector-sequencer/).

## Code Style

- Use Lua for all new files
- Blank lines should have no whitespace characters

## Security

- Never commit API keys or secrets
